# Reference: File Intake

The monthly reporting workflow should accept Amazon-native downloaded filenames.

The user should not need to manually rename files before using the specialist.

The intake workaround is an `INBOX/` folder.

The user drops original Amazon downloads into `INBOX/`, then states the marketplace and reporting month in the request.

## Common file examples

```text
BusinessReport-6-22-26.csv
