# Sample File Names

Amazon downloads may arrive with names like:

```text
BusinessReport-6-22-26.csv
2026Apr1-2026Apr30CustomUnifiedSummary.pdf
```

The first file name may reflect the download or report generation date. It should not be treated as the reporting month by itself.

The second file name usually contains a visible reporting period, but the specialist should still check it against the user's requested month.

## Example user request

```text
Run the Amazon USA monthly report for April 2026.
```

Expected target:

```text
Marketplace: USA
Reporting month: April 2026
```
